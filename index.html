Introduction

BlockCypher runs the Main and Test Bitcoin and Litecoin block chains as well as the Uro block chain. We provide APIs and libraries to browse block chains, create transactions and multisig transactions and register callbacks to get notified on block chain events.

Using BlockCypher, complicated tasks can be done in just a few lines of code:

{
    "inputs": [{"addresses": [sourceAddr]}],
    "outputs": [{
      "addresses"   : [pubkey1, pubkey2, pubkey3],
      "script_type" : "multisig-2-of-3",
      "value"       : 250000,
    }]
  }

Please contact us if you have any question.
