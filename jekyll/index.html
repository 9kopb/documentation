BlockCypher runs the Bitcoin and Litecoin Main and Test block chains as well as the Uro block chain. We provide APIs and libraries to browse block chains, create standard and multisig transactions, and register callbacks to get notified on all block chain events.

Using BlockCypher, complicated tasks such as multisig can be accomplished in a few lines of code:

{
    "inputs": [{"addresses": [sourceAddr]}],
    "outputs": [{
      "addresses"   : [pubkey1, pubkey2, pubkey3],
      "script_type" : "multisig-2-of-3",
      "value"       : 250000,
    }]
  }

Please contact us if you have any questions or need a user token.
